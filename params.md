| 参数 | 说明 |
| --- | --- |
| duration | 时长 |

| image | 图片 |
| multirates  | 当前视频有多少分辨率选项可下载 |

| m3u8 | 解析包含m3u8 |

| title | 标题 |
| tvid | 视频tvid(如果有) |
